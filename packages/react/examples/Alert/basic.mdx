## <i></i>基础样式

用来显示【成功、失败、警示】警示提示。

```tsx live=true
<>
  <Alert type="error" closable={false}>
    这是一条失败的提示信息。
  </Alert>

  <Alert type="warning" closable={false}>
    这是一条警告的提示信息。
  </Alert>

  <Alert type="info" closable={false}>
    这是一条普通的提示信息。
  </Alert>

  <Alert type="success" closable={false}>
    这是一条成功的提示信息。
  </Alert>
</>
```
