<>
  <Button
    type="primary"

  </Button>
  <Button onClick={() => message({ type: "success", message: "success" })}>
    success

    type="warning"

## 手动调用

```tsx live=true
<Button
  onClick={function popMessage() {
    message({
      type: "success",
      message: "^_^"
    });
  }}
>
  打开success弹窗
</Button>
```
