## 自定义色彩

多色彩标签以供不同场景使用。

- 可以使用我们准备好的颜色。

```tsx live=true
<>
  <Tag color="blue" style={{ margin: "0 5px" }}>
    blue
  </Tag>
  <Tag color="green" style={{ margin: "0 5px" }}>
    green
  </Tag>
  <Tag color="orange" style={{ margin: "0 5px" }}>
    orange
  </Tag>
  <Tag color="red" style={{ margin: "0 5px" }}>
    red
  </Tag>
  <Tag color="purple" style={{ margin: "0 5px" }}>
    purple
  </Tag>
  <Tag color="grey" style={{ margin: "0 5px" }}>
    grey
  </Tag>
  <Tag color="cyan" style={{ margin: "0 5px" }}>
    cyan
  </Tag>
  <Tag color="magenta" style={{ margin: "0 5px" }}>
    magenta
  </Tag>
</>
```

- 也可以自定义字体颜色。

```tsx live=true
<>
  <Tag color="#000098" style={{ margin: "0 5px" }}>
    #000098
  </Tag>
  <Tag color="#ec0aa0" style={{ margin: "0 5px" }}>
    #ec0aa0
  </Tag>
  <Tag color="#666" style={{ margin: "0 5px" }}>
    #666
  </Tag>
  <Tag color="#fad908" style={{ margin: "0 5px" }}>
    #fad908
  </Tag>
</>
```
