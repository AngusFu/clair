## Props

| Name               | Description                                                                   | Type                                                                                                                                                                                           | Required | Default                                                                                                                               |
| ------------------ | ----------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| content            | 气泡内容                                                                      | `React.ReactNode`                                                                                                                                                                              | `true`   | none                                                                                                                                  |
| visible            | 是否显示，可由 props 主动控制，不设置时，组件内部 state 控制                  | `boolean`                                                                                                                                                                                      | `false`  | none                                                                                                                                  |
| onVisibilityChange | 显示改变时候的回调                                                            | `({ detail: { visible: boolean; source: "hover"` \| `"click"` \| `"focus"` \| `"away" } }) => void` （"away" 是指点击非触发元素，且非弹出元素区域）                                            | `false`  | none                                                                                                                                  |
| placement          | 悬浮位置                                                                      | `"top"` \| `"right"` \| `"bottom"` \| `"left"` \| `"top-left"` \| `"top-right"` \| `"right-top"` \| `"right-bottom"` \| `"bottom-right"` \| `"bottom-left"` \| `"left-bottom"` \| `"left-top"` | `false`  | `"top"`                                                                                                                               |
| showDelay          | 显示延迟                                                                      | `number`                                                                                                                                                                                       | `false`  | `0`                                                                                                                                   |
| hideDelay          | 隐藏延迟                                                                      | `number`                                                                                                                                                                                       | `false`  | `100`                                                                                                                                 |
| trigger            | 触发方式                                                                      | `"hover"` \| `"click"` \| `"focus"` \| `Array<"hover"` \| `"click"` \| `"focus">`                                                                                                              | `false`  | `["hover", "focus"]`                                                                                                                  |
| footer             | 底部按钮区域                                                                  | `React.ReactNode`                                                                                                                                                                              | `false`  | 确认按钮默认是 `<Button type="primary" size="small">确认</Button>` 取消按钮默认是 `<Button type="default" size="small">取消</Button>` |
| onConfirm          | 确认回调，当 `confirmNode` 是 `React.Element` 时，确保能够设置 `onClick` 事件 | `(e: React.SyntheticEvent) => void`                                                                                                                                                            | `false`  | `() => void 0`                                                                                                                        |
| onCancel           | 取消回调，当 `cancelNode` 是 `React.Element` 时，确保能够设置 `onClick` 事件  | `(e: React.SyntheticEvent) => void`                                                                                                                                                            | `false`  | `() => void 0`                                                                                                                        |
