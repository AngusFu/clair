## 点击回调函数

### onConfirm

```tsx live=true
<PopConfirm content="Are you sure?" onConfirm={() => alert("yes")}>
  <Button>confirm</Button>
</PopConfirm>
```

### onCancel

```tsx live=true
<PopConfirm content="Are you sure?" onCancel={() => alert("cancel")}>
  <Button>cancel</Button>
</PopConfirm>
```
