## Props

| Name              | Description              | Type                                                           | Required | Default    |
| ----------------- | ------------------------ | -------------------------------------------------------------- | -------- | ---------- |
| defaultChecked    | 初始是否选中             | `boolean`                                                      | `false`  | `false`    |
| checked           | 指定当前是否选中         | `boolean`                                                      | `false`  | `false`    |
| disabled          | 是否禁用                 | `boolean`                                                      | `false`  | `false`    |
| checkedChildren   | 打开时的内容             | `ReactNode`                                                    | `false`  | -          |
| unCheckedChildren | 关闭时的内容             | `ReactNode`                                                    | `false`  | -          |
| size              | 开关大小                 | `"normal"` \| `"small"`                                        | `false`  | `"normal"` |
| checkedColor      | 打开时的背景色           | `string`                                                       | `false`  | -          |
| unCheckedColor    | 关闭时的背景色           | `string`                                                       | `false`  | -          |
| onChange          | 状态发生变化时的回调函数 | `({ target: { value: boolean }, nativeEvent: Event }) => void` | `false`  | -          |
